CSS Blending Modes

March 3rd

With the advent of CSS3, many new features are available to web developers that designers have been using in their work for years. One of the most crucial of these is the set of blending modes familiar to users of Photoshop or Illustrator. These blending modes are quite easy to implement with the the attribute: background-blend-mode: mode;

The modes available are nearly as varied as those in Adobe software, including: normal, multiply, screen, overlay, darken, lighten, color-dodge, color-burn, hard-light, soft-light, difference, exclusion, hue, saturation, color, and luminosity.

Pure CSS Dropdown Menu

January 15th

In working on my portal page, I wanted to create an uncluttered navigation menu that would intuitively house the many items that will eventually need to be accessed through it. I came across this guide to create such a navigation system without using Javascript. Though it provides a very clear and complete method for doing so, as an educational tool the guide falls short. The author fails to differentiate which CSS elements control the function of the navigation menu and which are merely part of the heavy styling they have applied. I managed to glean the key function elements as being:

nav ul ul{ display: none; }
and
nav ul li:hover > ul{ display: block; }

Though I have yet to incorporate this function into my own code, I'm sure the underlying principle has far greater application than simply creating menu systems.

GitHub

January 8th

Though it's primarily targeted at software developers, GitHub is an excellent resource for web designers as well. Those familiar with the Git source code management system may find the minimalist interface (at least on OSX) to be a more intuitive method to push assets to the server than more file-system oriented FTP clients. Its branching capabilities allow designers to work on multiple versions of the same project and then merge the best aspects of each together at the end of development, which can greatly benefit the learning process. However, the most helpful aspect for aspiring designers is the large and very active community wherein thousands of developers have made their source code viewable to the public and participate in discussions about their projects and techniques. GitHub also gives its users a free space to host and demo their works in progress in order to receive feedback from the community at large.
