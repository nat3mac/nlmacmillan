Syntax

Like any coding language, PHP has a specific set of punctuation marks and formatting guidelines that set it apart from other languages and let the interpreter process the code as PHP markup. For the most part, it retains key features of other related coding languages. Statements open and close with inequality brackets like in HTML, and value strings are ended with a semicolon, such as in CSS and Javascript. PHP is very similar to Javascript in nature, since they are both scripting languages; great control is offered thanks to the ability to write conditional statements - the basis of all programming.
